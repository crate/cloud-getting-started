=================================================
Create User Account and Install CrateDB Croud CLI
=================================================

In this section we will show you how to:

* Create your CrateDB Cloud user account
* Verify your CrateDB Cloud user account
* Log in to CrateDB Cloud for the first time
* Install the Croud CLI

.. Note::

   To interact with CrateDB Cloud you can also use `CrateDB Cloud Console`_
   that is our hosted web administration interface.
